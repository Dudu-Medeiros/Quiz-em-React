# 🎯 DevQuiz

Um quiz interativo desenvolvido em **React** para testar conhecimentos em **HTML**, **CSS** e **JavaScript** de forma dinâmica e divertida.

## ✨ Funcionalidades

* 📝 Perguntas de múltipla escolha.
* ⚡ Feedback instantâneo das respostas.
* 📊 Exibição da pontuação ao final.
* 🔄 Possibilidade de reiniciar o quiz.
* 🎨 Interface responsiva e intuitiva.

## 🛠️ Tecnologias

* React
* JavaScript (ES6+)
* HTML5
* CSS3

## 🚀 Objetivo

O projeto foi criado para praticar conceitos de **React**, como componentes, estados, renderização condicional e gerenciamento de eventos, ao mesmo tempo em que oferece uma forma interativa de revisar fundamentos do desenvolvimento web.
